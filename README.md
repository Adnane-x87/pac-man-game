# Pac-Man Game

This repository contains a classic Pac-Man game implemented using HTML, CSS, and JavaScript.

## Description

This project is a web-based rendition of the iconic arcade game, Pac-Man. Players navigate Pac-Man through a maze, eating dots while avoiding four colored ghosts. The game features traditional mechanics, including scoring, lives, and increasing difficulty across levels.

## Features

*   **Classic Gameplay**: Faithful recreation of Pac-Man's core mechanics.
*   **Maze Navigation**: Control Pac-Man's movement using arrow keys or W, A, S, D.
*   **Ghosts**: Four distinct ghosts (Blinky, Pinky, Inky, Clyde) with basic AI for pursuit.
*   **Food Collection**: Eat all dots in the maze to advance to the next level.
*   **Scoring System**: Points awarded for eating dots.
*   **Lives System**: Multiple lives before game over.
*   **Collision Detection**: Accurate collision handling for Pac-Man, ghosts, and walls.
*   **Responsive Design**: Basic styling for a playable experience in a web browser.

## Technologies Used

*   **HTML5**: For structuring the game's web page.
*   **CSS3**: For styling the game elements and layout.
*   **JavaScript (ES6+)**: For game logic, animation, and interactivity.

## How to Play

1.  Open `index.html` in your web browser.
2.  Use the **Arrow Keys** (Up, Down, Left, Right) or **W, A, S, D** keys to move Pac-Man.
3.  Eat all the dots on the screen while avoiding the ghosts.
4.  If a ghost catches Pac-Man, a life is lost. Lose all lives, and it's game over!
5.  Clear the maze of all dots to proceed to the next level.
